# RL training scripts for terminal agent


## Run instructions

1. Convert terminal bench like tasks folder to train / test dataset using `convert_tasks_to_dataset.py`

2. Run `run_terminal_agent_flow.py` to validate the tasks and agents are setup correctly

3. Run `train_terminal_agent_flow.sh` to start training

**Note**: These scripts serve as an example of integration camel agent to rLLM framework and have not been extensively tested yet.
